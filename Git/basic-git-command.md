#### 1. To intialise git repo
`git init`

#### 2. Add file for tracking
`git add filename`

#### 3. Commit changes 
`git commit -m "message"`

#### 4. Check current changes/status
`git status`

#### 5. To check commit history 
`git log`

#### 6. To commit latest changes into last commit 
`git commit --amend`
